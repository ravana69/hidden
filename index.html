Get A Free VPS

Yeah you heard it right, you can get a free vps that will have A Quad-core CPU with 16GB RAM. which will look something like this

As you can tell by looking at the picture above, we will use cloud shell for this.

If you want to ask something or having a problem related to this, You can ask here

And if you're sharing this with your friends, don't forget to give me credit lol.

Requirements

  1. A Google account.
  2. A device like phone or laptop or something.
  3. patience.

Steps

  1. Open https://shell.cloud.google.com
  2. A terminal will pop up. see the image below.

  1. Now you have to execute some commands in the terminal you opened in step 2.
  2. sudo apt-get update
  3. sudo apt-get install xfce4 xfce4-terminal To install xfce4 desktop enviroment.
  4. Now go to https://remotedesktop.google.com/headless
  5. Click on begin.

  1. Copy the second link on that page.

  1. Now you have to execute some more commands on the terminal you opened on step 2.
  2. wget <the_link_you_copied

  1. sudo dpkg -i chrome* [This will fail! but don't worry it will be fixed in next step]

  2. sudo apt-get install -f this will fix the CRDP installation.

  3. Click Next in the page you opened in step 6.

  1. Click on Authorize.

  1. Choose an account.

  1. Copy the third code on that page.

  1. Paste the code on cloud-shell terminal.

  1. It will ask for a pin, enter anything like 123456.

  2. Now Click on Remote access.

  1. Click on your session.

  1. It will ask for password, enter the password you created in step 18.

  2. Then a window will popup, choose xfce and click ok.

  1. Xfce4 will open, Click on default panel and you're ready to go.

Note

  1. Server will only work as long as the cloud shell terminal is opened , (install chrome on rdp and open the shell terminal there to fix).
  2. Server will only work 50 hours/week, so get 2-3 google accounts if you want to always use it.
  3. If you restarted the server, you need to do all the steps again.
  4. Contact https://t.me/patheticprogrammers for help.
